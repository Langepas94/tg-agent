# Руководство пользователя

Это основной источник правил использования Telegram-бота `tg-agent` и ответов
службы поддержки. Если пользовательская и техническая документация описывают
ситуацию по-разному, поддержка следует этому разделу, а технические документы
использует только для диагностики.

## Быстрый маршрут

1. [Первый запуск и доступ](getting-started.md) — как открыть бота, пройти
   авторизацию и безопасно обратиться за помощью.
2. [Планирование отдыха](trip-planning.md) — основной пользовательский сценарий

3. [Команды и память](commands-and-memory.md) — профиль, предпочтения, очистка

4. [Поддержка и частые вопросы](support-and-faq.md) — самостоятельные проверки и
   данные, которые нужно передать оператору.

## Что умеет бот

Бот помогает выбрать формат активного отдыха с учетом дат, места старта,
погоды и ограничений группы. Он сравнивает варианты, просит выбрать подходящий
и только после выбора строит конкретный маршрут. Для прогноза, карт и внешних
действий используются подключенные инструменты; если достоверных данных нет,
бот должен прямо сообщить об этом.

Бот не заменяет официальные предупреждения экстренных служб, правила посещения
территорий и личную оценку рисков. Перед поездкой перепроверьте прогноз,
доступность маршрута, транспорт и необходимое снаряжение.
